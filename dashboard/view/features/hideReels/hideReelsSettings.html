Hide Reels Configuration

Customize how Reels are hidden and blocked on Instagram

Disable Reels Page

When enabled, accessing individual Reels (/reels/{id}) will show a blocked screen instead of the reel content.

Disable Profile Reel Tab

When enabled, the Reels tab on profile pages will be disabled and grayed out, preventing access to profile reels.

How It Works

About Hide Reels

  • Reels navigation links are always hidden when the feature is enabled
  • All sub-features work dynamically without requiring a page refresh
  • Settings sync across all open Instagram tabs automatically
  • When the main feature is disabled, all sub-features are also disabled
