Disable Inbox Page

When enabled, accessing /direct will show a blocked screen instead of the inbox. You'll be redirected to the homepage.

Hide Floating Button

Hide the floating chat button that appears on Instagram pages.

Disable Message to Profile

Disable the "Message" button on user profiles to prevent sending direct messages.

How It Works

About Hide Direct Message

  • Direct message links in navigation are always hidden when the feature is enabled
  • All sub-features work dynamically without requiring a page refresh
  • Settings sync across all open Instagram tabs automatically
  • When the main feature is disabled, all sub-features are also disabled
