Hide Stories Configuration

Customize how Stories are hidden on Instagram

Disable Stories Page

When enabled, accessing /stories will show a blocked screen instead. You'll be redirected to the homepage.

Hide all Profile Stories

Hide story rings around profile pictures across Instagram. Clicking on profile pictures will still work, but story viewing will be disabled.

How It Works

About Hide Stories

  • Story tray is always hidden when the feature is enabled
  • All settings work dynamically without requiring a page refresh
  • Settings sync across all open Instagram tabs automatically
  • When the main feature is disabled, all sub-features are also disabled
